# Setup Flow

A Next.js project that shows how to build a multi-step workspace setup form with sidebar navigation using [Formity](https://formity.app).

The form has a sidebar that lists all steps and lets users jump to any of them at any time.

## What you'll learn

You'll see how to use the `jump` element to make each step addressable by ID, and how to wire a sidebar component to Formity via a ref so it can trigger navigation from outside the form itself.

## Getting started

Install the dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
